广汽全新雷凌配置曝光 共推十款车型

缩小字体 放大字体 收藏 微博 微信 分享
腾讯QQ QQ空间

广汽丰田全新雷凌部分配置信息曝光，新车将推出搭载1.8L混动和1.2T两种动力共计4款车型，分别为双擎科技版（1.8L混动）、双擎尊享版（1.8L混动）、185T科技版（1.2T）、185T尊享版（1.2T）。

　　新浪汽车讯 日前，广汽丰田全新雷凌部分配置信息曝光，新车将推出搭载1.8L混动和1.2T两种动力共计4款车型，分别为双擎科技版（1.8L混动）、双擎尊享版（1.8L混动）、185T科技版（1.2T）、185T尊享版（1.2T）。

　　安全层面，全新雷凌标配驾驶席/副驾驶席二级式正面SRS空气囊、驾驶席膝部SRS空气囊、前排侧SRS空气囊、侧窗帘式SRS空气囊、副驾驶席坐垫SRS空气囊、倒车影像、DRCC动态雷达巡航控制系统、车辆防盗警报（APP）、远程故障诊断服务、PCS预碰撞安全系统、紧急救援/道路救援、被盗追踪系统（远程启动限制）、胎压监测、LTA车道循迹辅助系统（带低速跟车行驶功能）、疏忽提醒、保养提醒（车辆状况检测服务）。

　　动力方面，全新雷凌燃油版和双擎版分别搭载的是1.2T直列四缸发动机和一套1.8L混合动力系统。其中，1.2T发动机最大功率116马力，峰值扭矩185牛·米，传动系统匹配CVT无级变速箱；而双擎车型的混动系统由1.8L阿特金森循环自然吸气发动机、电动机和混动系统专属的E-CVT变速箱组成，其中发动机最大功率99马力，系统综合最大功率或为136马力。
