达斯汀墨西哥世锦赛赢美巡20胜 伍兹T10李昊桐T19

达斯汀墨西哥世锦赛赢美巡20胜 伍兹T10李昊桐T19
2019年02月25日 07:30 新浪体育
缩小字体 放大字体 收藏 微博 微信 分享 0
腾讯QQ QQ空间
达斯汀-约翰逊夺冠

　　北京时间2月25日，世锦赛-墨西哥锦标赛在查普特佩克高尔夫俱乐部落下大幕，世界第三达斯汀-约翰逊以5杆的巨大优势夺冠，泰格-伍兹并列第10，李昊桐排名并列19位。

　　达斯汀-约翰逊决赛轮延续了前三轮的强势，仅在3号洞吞下当日唯一一个柏忌，随后抓到6只小鸟，打出66杆（-5），四轮总成绩263杆（64-67-66-66），低于标准杆21杆，毫无悬念的赢得冠军。

　　这是达斯汀-约翰逊的第20个美巡赛冠军，也是他个人的第6个世锦赛冠军头衔。同时，这也是达斯汀在墨西哥锦标赛的第3冠，两年前，赛事第一次搬迁到墨西哥城的时候，他便取得胜利，而2015年赛事在多拉举行的时候，他也赢得了一次。

　　“拿到美巡20胜真的很难，能够在步入35岁之间达到这一成绩令人难以置信。”达斯汀说道。

　　与达斯汀同组的北爱尔兰名将罗里-麦克罗伊未能给他带来威胁，小麦打出67杆（-4），总成绩低于标准杆16杆，排名单独第二位。

麦克罗伊未能给达斯汀-约翰逊造成威胁

　　而同组的另外一人、美国大师赛卫冕冠军帕特里克-瑞德则表现令人失望，他打出73杆（+2），排名滑落至并列14位。

　　保罗-卡西（65杆）、伊恩-保尔特（68杆）、奇拉德克-阿菲巴拉特（68杆），同以低于标准杆11杆的总成绩，排名并列第三位。

　　前三轮发挥平平的贾斯汀-托马斯终于在决赛轮觉醒，他打出10鸟1柏忌的单轮最低62杆，总成绩-9杆，排名升至单独第九位。

伍兹收获前十

　　泰格-伍兹出门连抓两只小鸟，后又有1鸟1柏忌进账，以69杆作收，总成绩-8杆，排名并列第10位。

李昊桐决赛轮记分卡

　　“中国一哥”李昊桐决赛轮发挥出色，他抓到6只小鸟，吞下2个柏忌，打出67杆（-4），总成绩低于标准杆5杆，排名并列19位。
