原创 雷军：小米9供货超150万台 米粉节备货充足

2019-04-01 16:40 来源:搜狐科技视界 雷军 /小米 /科技

原标题：雷军：小米9供货超150万台 米粉节备货充足

搜狐科技/杨锦

为了给今年的米粉节造势，4月1日下午，小米在北京举行“小米9周年米粉节 中国区总裁见面会”，小米中国区总裁王川提前预告了米粉节上的优惠活动，以及将要在米粉节首销的新品。

会上，王川透露，小米9系列出货量已经突破150万台，红米Redmi Note7系列累计出货超过400万台。他表示，“小米9应该是产能爬坡速度最快的小米手机之一，当前产能远超内部预期。”

同时，雷军也在微博上发布了这一消息，并补充说“小米9/小米9 SE供货超过150万台，其中，小米9超过100万台。”雷军宣布，小米9系列手机在米粉节备货充足。

小米9是今年2月20日小米推出的旗舰系列新机，售价2999元起。前不久，因为缺货问题遭到网友吐槽，雷军当时表示，“假如小米9系列首月供货不能超过百万台，我亲自到工厂拧螺丝！”

当前小米中国区总裁王川2012年加入小米，是小米集团第8位联合创始人，此前在小米主要负责电视业务。2018年12月13日，小米集团发内部邮件宣布人员架构调整，任命集团高级副总裁王川兼任中国区总裁，向CEO汇报。

今年米粉节，小米将拿出多款手机及智能产品，包括小米9、小米9王源限量版、小米8屏幕指纹版、小米MIX 3、小爱触屏音箱触屏版等参与活动。此外，小米还和LINE FRIENDS合作，推出了小米9 SE布朗熊限量套装等产品，这些产品均将于4月9日米粉节首销。
